# Brother CNC 设备端配置

本指南介绍如何配置 Brother CNC 机床的通信和网络参数，以便 Neuron Brother CNC 插件通过 TCP/IP 连接并采集数据。

## 准备工作

开始配置前，请确保以下条件已满足：

- 可以操作 Brother CNC 控制面板。
- 拥有修改机床通信参数的权限。
- 机床已通过网线连接到网络。
- 已准备好机床的静态 IP 地址、子网掩码和网关。

## 配置网络和通信参数

### 步骤一：进入通信参数页面

1. 按下面板上的**数据库**按钮。
2. 按下 **F6**（通信参数），选择通信参数设置。

### 步骤二：配置远程访问参数

在通信参数页面中，配置以下参数：

| 参数 | 设置 | 说明 |
| ----------------- | ---- | ------------------------------------ |
| **数据改写（从属）** | 是 | 允许外部设备从 CNC 读取数据。 |
| **远程操作** | 有效 | 启用机床的远程访问功能。 |

### 步骤三：配置网络参数

设置机床的网络参数：

| 参数 | 值 |
| ---------- | ---------------------------------------- |
| **IP 地址** | 输入期望的静态 IP 地址。 |
| **网关** | 输入网络网关地址。 |
| **子网掩码** | 输入子网掩码（如 255.255.255.0）。 |

::: tip
IP 地址需与 Neuron 主机处于同一子网，默认通信端口为 **10000**。
:::

### 步骤四：启用以太网访问

将**限制以太网访问**参数从"是"改为"否"，以允许外部 TCP/IP 连接。

### 步骤五：保存设置

1. 修改完所有参数后，按下**完成模式**。
2. 再按下**退出编辑**保存配置。

## 验证网络连通性

配置完成后：

1. 记录 Brother CNC 上配置的 IP 地址。
2. 从 Neuron 主机 ping Brother CNC 的 IP 地址，验证基本网络连通性：
   ```shell
   ping <BROTHER_CNC_IP_ADDRESS>
   ```
3. 确认 Neuron 主机能够访问 TCP 端口 **10000**。

验证通过后，在 Neuron 端配置 Brother CNC 插件。详情请参考 [Brother CNC](./brother-cnc.md)。

## 故障排查

### 无法连接到 Brother CNC

**可能原因：**
- Neuron 中配置的 IP 地址不正确。
- 限制以太网访问仍设置为"是"。
- 数据改写设置为"否"。
- 远程操作设置为"无效"。
- 防火墙阻止了 TCP 端口 10000。
- 机床与 Neuron 主机不在同一网段。

**解决方法：**
1. 确认 Brother CNC 上配置的 IP 地址。
2. 确认上述步骤三和步骤四中的网络参数已正确设置。
3. 检查限制以太网访问是否已设为"否"。
4. 确认数据改写为"是"、远程操作为"有效"。
5. 确认防火墙未阻止 TCP 端口 10000。
6. 从 Neuron 主机 ping Brother CNC IP 地址，验证网络连通性。

### 重启后配置丢失

**可能原因：**
- 参数保存操作未正确完成。

**解决方法：**
1. 重新进入通信参数页面。
2. 检查各项设置是否仍在。
3. 确保修改参数后，依次按下**完成模式**和**退出编辑**保存。
4. 必要时重启机床并重新检查设置。
